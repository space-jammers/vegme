# Vegme

VegMe is a vegan and vegetarian recipe search engine powered by Edamam Recipe API.


## Built With

* Rspec
* Devise
* Codeship
* HTTParty
* Rubocop
* Foundation
* Figaro
* Better Errors
* SimpleCov
* Chart-js
* Simple Form
* Animate.css
* Will Paginate
